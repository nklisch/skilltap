# Windsurf/Devin Desktop Cascade MCP

Cascade documents direct MCP configuration through
`~/.codeium/windsurf/mcp_config.json`, supporting stdio, HTTP, and SSE. The
page exposes enablement and tool selection in the desktop interface. It does
not document a project-scoped MCP file parallel to its workspace skill path.

## Key passages

- “mcp_config.json” names the global file and JSON schema.
- “Adding a new MCP” and “Configuring MCP tools” describe UI state.
- The transport section covers stdio, streamable HTTP, SSE, and OAuth.
- No workspace/project MCP path is identified on the page.
