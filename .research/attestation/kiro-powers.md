# Kiro powers

Kiro Powers bundle `POWER.md`, MCP configuration, and optional hooks/steering. The current guide describes one-click installation in the IDE or web, with GitHub sharing; it explicitly says powers are currently available in the IDE and planned for the CLI. This is not a CLI skill/plugin lifecycle candidate.

## Key passages

- “What's in a power?” defines a bundle unlike the `SKILL.md` contract.
- “One-click install” is a web/IDE action.
- The FAQ states CLI availability is planned rather than current.
