# Kilo Code skills

Kilo supports global/project `.kilo/skills` and compatibility `.agents/skills`, plus remote skill indexes configured through `skills.urls`. Current CLI documentation describes discovery and reload, but not native install/update/remove commands for skills.

## Key passages

- “Global Skills” and “Project Skills” define separate scopes.
- “Compatibility Directories” lists `.agents/skills`.
- “Additional Skill Paths and Remote URLs” requires a remote `index.json`; the page says the new platform has no marketplace UI yet.
