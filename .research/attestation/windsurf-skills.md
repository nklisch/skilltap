# Windsurf/Devin Desktop Cascade skills

Cascade defines skills as directories containing `SKILL.md` and supporting
resources. Workspace skills live in `.windsurf/skills`; global skills live in
`~/.codeium/windsurf/skills`; `.agents/skills` is also discovered. The full
skill and siblings load on demand.

## Key passages

- “Manual Creation” names workspace and global paths.
- “Adding Supporting Resources” shows sibling templates and checklists.
- “Skill Scopes” documents `.agents/skills` compatibility.
- “Invoking Skills” describes automatic and explicit loading.
