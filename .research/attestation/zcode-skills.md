# ZCode skills

ZCode defines a skill as a directory containing `SKILL.md`. User skills live
under `~/.zcode/skills`. Its import workflow can create either a symlink or a
copy and target global or current-workspace scope; imported skills remain
individually observable and enableable in settings.

## Key passages

- “Create A Skill” defines the directory and user path.
- “Import Skills From An External Agent” documents copy and symlink modes.
- The import target can be Global or current Project.
- “Manage Skills” exposes source, description, and enabled state.
