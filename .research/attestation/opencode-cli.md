# OpenCode CLI

OpenCode has an `opencode plugin <module>` command that installs a plugin and updates config, with global and force flags. The current command reference does not document native plugin list, update, uninstall, or marketplace-registration commands.

## Key passages

- The `plugin` command is described as “Install a plugin and update your config.”
- The command index contains no corresponding plugin lifecycle group beyond install.
