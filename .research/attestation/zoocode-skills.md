# Zoo Code skills

Zoo Code resolves skills from project and global `.roo` and `.agents`
directories, including mode-specific roots. It loads the main `SKILL.md` and
allows explicitly referenced sibling files. Project definitions take priority
over global definitions.

## Key passages

- “How It Works” lists all project and global search locations.
- “Skill Loading” identifies `SKILL.md` as the main instruction file.
- “Linked Files” states sibling files remain available for explicit reads.
- “Key Features” documents project override precedence.
