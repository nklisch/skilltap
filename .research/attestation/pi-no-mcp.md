# Pi has no built-in MCP client

Pi's current product page explicitly lists “No MCP” among features intentionally
omitted from the core harness. It directs users to build or install an extension
that adds MCP support. The same page separately documents that extensions and
skills can be bundled into installable Pi packages.

## Key passages

- “What we didn't build” includes “No MCP.”
- The accompanying guidance says to build an extension that adds MCP support.
- “Primitives, not features” describes third-party extensions and packages as
  the mechanism for adding omitted capabilities.
