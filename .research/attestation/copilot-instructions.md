# GitHub Copilot CLI instructions

Copilot CLI supports repository-wide `.github/copilot-instructions.md`, path-specific instruction files, and `AGENTS.md`. It also accepts `CLAUDE.md` or `GEMINI.md` at repository root and a personal `$HOME/.copilot/copilot-instructions.md`. `COPILOT_CUSTOM_INSTRUCTIONS_DIRS` adds searched locations.

## Key passages

- The “Agent instructions” section names `AGENTS.md` and additional configured directories.
- The “Local instructions” section gives the personal home file.
- The same page documents repository-wide and path-specific instruction surfaces.
