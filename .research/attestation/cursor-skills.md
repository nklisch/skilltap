# Cursor Agent Skills

Cursor's 2.4 release introduced Agent Skills in both the editor and CLI. It
describes skills as `SKILL.md` definitions capable of including commands,
scripts, and instructions. The release page does not itself enumerate stable
global/project skill paths, so an adapter must verify the current filesystem
contract before mutation.

## Key passages

- “Skills” states support in editor and CLI.
- The same section says `SKILL.md` definitions may include commands, scripts,
  and instructions.
- The page links to further documentation but does not state load paths.
