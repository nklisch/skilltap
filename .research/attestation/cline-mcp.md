# Cline MCP

Cline CLI documents MCP configuration in user-level `~/.cline/mcp.json`. Its
wizard can list, add, edit, enable/disable, and delete servers, and `cline
config mcp --json` provides non-interactive observation. The page does not
document a project-scoped MCP file for CLI or the IDE extension.

## Key passages

- “Manual config” names the CLI user file.
- “CLI MCP wizard” lists lifecycle operations.
- “CLI” documents JSON observation.
- No project MCP file is identified on the page.
