# Kiro CLI commands

Kiro's CLI reference documents agent, MCP, update, diagnostics, integration,
and settings commands. It exposes a `KIRO_HOME` override for global agents,
prompts, skills, steering, settings, and sessions, but does not document a
Powers or skill package install/update/remove lifecycle.

## Key passages

- The command index includes `kiro-cli mcp add|remove|list` and `kiro-cli update`.
- The environment-variable section defines `KIRO_HOME` as the global home for
  agents, prompts, skills, steering, settings, and sessions.
