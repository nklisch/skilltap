# Cline skills

Cline defines each skill as a directory containing required `SKILL.md` and
optional docs, templates, and scripts. Project skills load from `.cline/skills`
and compatibility directories; user skills load from `~/.cline/skills`.

## Key passages

- “Skill Structure” defines the complete directory.
- “Where Skills Live” names project and global roots.
- “Bundling Supporting Files” lists supported sibling resources.
- The UI exposes skill enablement without deleting the directory.
