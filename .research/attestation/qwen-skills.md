# Qwen Code skills

Qwen defines a skill as a directory containing `SKILL.md` plus optional scripts, references, and templates. Personal skills are under `~/.qwen/skills/`; project skills under `.qwen/skills/`; extension skills are under an extension's `skills/` directory. The guide documents YAML frontmatter requirements and update/removal as directory operations.

## Key passages

- “What are Agent Skills?” requires the complete directory and `SKILL.md`.
- “Personal Skills” and “Project Skills” provide separate user/project roots.
- “Extension Skills” says installed extensions can provide the same skill format.
