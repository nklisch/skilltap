# Goose skills

Goose discovers complete skill directories from global/project `.agents/skills` and plugin-provided locations. Its documentation directs users to install skills through the external `npx skills` CLI and enable Goose's Summon extension; Goose itself does not document a native marketplace/plugin lifecycle for these skills.

## Key passages

- “Skill Locations” lists global, project, and installed-plugin paths.
- “Installing a Skill” in the official tutorial uses `npx skills add` and a separate Summon extension toggle.
