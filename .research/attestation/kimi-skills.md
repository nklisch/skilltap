# Kimi Code Agent Skills

Kimi recommends directory-form skills with `SKILL.md` and sibling scripts,
references, and other resources. User roots include `$KIMI_CODE_HOME/skills`
and `~/.agents/skills`; project roots include `.kimi-code/skills` and
`.agents/skills`. Project definitions take priority. Changes are picked up in a
new session.

## Key passages

- “Creating a Skill” distinguishes the recommended directory form from flat files.
- “Skill Locations” lists user and project roots and their precedence.
- The complete example stores `references/checklist.md` beside `SKILL.md`.
- The example says to start a new session after installing the skill.
