# Roo Code shutdown

Roo Code's official documentation states that the Roo Code extension was shut
down on May 15, 2026. It directs users to Zoo Code, a community fork, or Cline
as alternatives. Historical skills and MCP capabilities therefore do not make
Roo an appropriate new adapter target.

## Key passages

- The page banner states the extension was shut down on May 15.
- The same notice identifies Zoo Code and Cline as alternatives.
