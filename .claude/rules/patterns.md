# Patterns Index

- **Result type**: `Result<T,E>` discriminated union with `ok()`/`err()` constructors ‚Äî all fallible core functions return this, never throw ‚Üí [patterns/result-type.md]
- **Error hierarchy**: `UserError`, `GitError`, `ScanError`, `NetworkError` extend `SkilltapError` with optional `hint` field ‚Üí [patterns/error-hierarchy.md]
- **Zod boundary**: Zod schema is source of truth for type + validation; `safeParse` + `z.prettifyError` at every data ingestion point; use `.prefault({})` not `.default({})` for nested objects ‚Üí [patterns/zod-boundary.md]
- **Callback-driven options**: Core functions accept typed option objects with async callbacks (`onWarnings`, `onSelectSkills`, `onConfirm`, etc.) for UI decision points; omit = auto-proceed ‚Üí [patterns/callback-driven-options.md]
- **Policy composition**: `composePolicy(config, flags)` pure function ‚Üí `EffectivePolicy`; centralizes config + CLI flag precedence; returns `Result<>` for invalid combos ‚Üí [patterns/policy-composition.md]
- **Agent mode branching**: Commands fork early into `runAgentMode()` (plain text, auto-accept, hard-fail) vs `runInteractiveMode()` (clack spinners, prompts) ‚Üí [patterns/agent-mode-branching.md]
- **SourceAdapter strategy**: Plain object literals implementing `{ name, canHandle(), resolve() }`; resolver iterates `ADAPTERS[]` in priority order ‚Üí [patterns/source-adapter.md]
- **AgentAdapter strategy**: `AgentAdapter` interface with `detect()`/`invoke()`; factory functions (`createCliAdapter`, `createCustomAdapter`, `createOllamaAdapter`); priority resolution via `resolveAgent()` ‚Üí [patterns/agent-adapter-strategy.md]
- **Bun shell git**: All git via `wrapGit<T>(fn, msg, hint?)` wrapper + `extractStderr(e)` helper; returns `Result<T, GitError>`; `.quiet()` on all commands ‚Üí [patterns/bun-shell-git.md]
- **Config I/O**: Load = ensureDirs ‚Üí exists check ‚Üí read ‚Üí parse format ‚Üí safeParse ‚Üí ok; Save = ensureDirs ‚Üí serialize ‚Üí Bun.write ‚Üí ok ‚Üí [patterns/config-io.md]
- **Security detector composition**: 7 independent `(content: string) => PatternMatch[]` functions composed in a for-loop in `scanStatic()`; add new detectors by appending to the `detectors` array ‚Üí [patterns/security-detector-composition.md]
- **InstallResult with warnings**: `installSkill()` returns `{ records, warnings, semanticWarnings }`; `onWarnings`/`onSemanticWarnings` callbacks for per-skill interception; `skipScan: true` for tests ‚Üí [patterns/install-result-with-warnings.md]
- **Fixture repo factory**: `createX()` returns `{ path, cleanup }` ‚Äî copy static fixture dir, `initRepo`, `commitAll`; always `dot:true` in Bun.Glob.scan ‚Üí [patterns/test-fixtures.md]
- **Result test assertions**: `expect(result.ok).toBe(true)` then `if (!result.ok) return` guard; `VALID_*` constants with spread for schema variants ‚Üí [patterns/test-result-assertions.md]
- **CLI subprocess testing**: CLI integration tests use `Bun.spawn` with `SKILLTAP_HOME`/`XDG_CONFIG_HOME` env vars; `stdin: "pipe"` for non-TTY tests ‚Üí [patterns/cli-subprocess-testing.md]
- **Injectable dependencies**: Core functions with external I/O accept `_dep = realImpl` optional params; tests pass mocks as 2nd/3rd args; private `type Fn = typeof realFn` aliases enforce signature ‚Üí [patterns/injectable-dependencies.md]
- **Graceful fallback chain**: Optional verifiers return `T | null` (never `Result`); caller cascades through priority tiers; outer `try/catch` guarantees a valid result is always returned ‚Üí [patterns/graceful-fallback-chain.md]
- **Adapter-driven branching**: After `resolveSource()`, `resolved.adapter` (and `record.repo?.startsWith("npm:")` on update) gates distinct install/update/trust paths per source type ‚Üí [patterns/adapter-driven-branching.md]
- **Single-source definitions**: One authoritative constant per enumerable concept; all consumers import/derive ‚Äî never copy literal values. Agent metadata (`AGENT_PATHS`, `AGENT_LABELS`, `VALID_AGENT_IDS`) lives in `core/src/symlink.ts`; config enum arrays (`SCAN_MODES`, etc.) are exported constants in `core/src/schemas/config.ts` ‚Üí [patterns/single-source-definitions.md]
